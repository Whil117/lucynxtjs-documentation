# Components

Exist a lot of components for you. You can combine with other atoms without problems.
All Components has the same props if you used html only. But with new properties such:

CustomCSS, customCSSLabel, customCSSOption, customCSSError, css.

These properties recives a function and should return a css

You can use function css from Emotion

```js
import { css } from "@emotion/react";
```

## AtomText

It's a great component because you have the typical properties for example: width, textAlign, textDecoration and more as params. This Component contains customCSS if you want to more customization

```javascript
<AtomText
  customCSS={(css) => css`
    background-color: red;
  `}
>
  Hello Lucy
</AtomText>
```

export const ExampleTextAtom = () => {
return (

<AtomText
customCSS={(css) => css`    background-color: red;
 `}

> Hello Lucy

</AtomText>
) };

## AtomWrapper

The best component, You always going to work with this atom, this atom is a special of container or div. With CustomCSS you have a powerfull container
Compatible with flex and grid and draggable container provide by Framer Motion

```js
<AtomWrapper gap="5px" draggable>
  <AtomWrapper draggable backgroundColor="#07deff" padding="5px">
    <AtomText>Default Text</AtomText>
  </AtomWrapper>
  <AtomWrapper
    draggable
    customCSS={(css) => css`
      border: 1px solid #07deff;
    `}
    padding="5px"
  >
    <AtomText color="white">Default Text</AtomText>
  </AtomWrapper>
</AtomWrapper>
```

export const ExampleWrapper = () => {
return (
<AtomWrapper gap="5px" draggable>

<AtomWrapper draggable backgroundColor="#07deff" padding="5px">
  <AtomText>Default Text</AtomText>
</AtomWrapper>
<AtomWrapper draggable customCSS={(css)=> css`
border:1px solid #07deff;
`} padding="5px">
  <AtomText color="white">Default Text</AtomText>
</AtomWrapper>
</AtomWrapper>
) };

## AtomIcon

If you want to render a svg, you can use with a external url, This is great because you can chache the asset and improve the page

You have more options as: color fill by default black, width, height. with a customCSS

This Atom use swr library for fetching the icon and embed the svg inside a dangerouslySetInnerHTML with a AtomWrapper

```js
<AtomIcon
  src="https://res.cloudinary.com/whil/image/upload/v1679173748/GITHUB_udwkxu.svg"
  color="#0072FF"
/>
```

## AtomImage

Typically a web page use images, using a external URL image. you can render a imagen with a lot of properties as: ObjectFit, width, Height, customCSS and more. This component has cache and improve the page
